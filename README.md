# Bíblia & Harpa Cristã

Um aplicativo de Bíblia Sagrada e Harpa Cristã desenvolvido com tecnologias modernas para Android. O foco principal é a velocidade e a acessibilidade, funcionando **100% offline**.

##  Funcionalidades

- **Bíblia Completa Offline:** Acesse todos os livros, capítulos e versículos sem precisar de internet.
- **Harpa Cristã:** Hinário completo integrado para louvores.
- **Marcação de Versículos:** Destaque seus versículos favoritos com cores personalizadas.
- **Customização de Leitura:** Ajuste o tamanho da fonte para uma leitura mais confortável.
- **Compartilhamento:** Compartilhe um ou vários versículos selecionados diretamente para suas redes sociais ou contatos.
- **Interface Moderna:** Desenvolvido inteiramente em **Jetpack Compose** com suporte a temas e navegação fluida.

##  Tecnologias Utilizadas

- **Linguagem:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Injeção de Dependência:** [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
- **Persistência de Dados:** [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) (Preferências e Configurações)
- **Arquitetura:** MVVM (Model-View-ViewModel)
- **Navegação:** Compose Navigation

##  Como instalar o App (APK)

Se você quer apenas testar o aplicativo no seu dispositivo Android, siga os passos abaixo:

1. Vá até a aba [**Releases**](https://github.com/MatheusMed/Biblia/releases/) deste repositório.
2. Procure pela versão mais recente (ex: `v1.0.0`).
3. Na seção **Assets**, baixe o arquivo `.apk`.
4. No seu celular, abra o arquivo baixado e permita a instalação de fontes desconhecidas se solicitado.
